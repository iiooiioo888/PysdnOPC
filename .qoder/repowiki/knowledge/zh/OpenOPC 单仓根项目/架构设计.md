本仓库是 OPC 产品的顶层聚合包：
- pyproject.toml 通过 [project.scripts] 将 opc 命令绑定到 opc.cli.app:main，作为 presentation 子模块对外暴露的唯一入口；skills 与 config_and_docs 子目录通过 [tool.hatch.build.targets.wheel.force-include] 映射为 opc/skills_assets/core 与 opc/config_templates，供运行时在包内加载。
- 所有子模块（presentation、skills、tests、config_and_docs）共享同一 Python 包命名空间 opc/，依赖由根 pyproject.toml 集中声明，避免各子包各自维护依赖。
- tests 子模块覆盖上述所有子模块的单元测试与 Playwright E2E，位于同一仓库便于一次性运行。
