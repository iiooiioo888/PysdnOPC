- 对外部可选依赖采用 try/except ModuleNotFoundError + 延迟 _require_* 函数模式，使包在无 mcp 安装时仍可导入但调用时给出清晰错误。
- 异步资源生命周期统一通过 `AsyncExitStack` 管理，`start` 中 enter 上下文、`stop` 中 aclose，确保异常路径也能正确释放连接。
- 远程连接实现先尝试 StreamableHTTP，失败后重置 AsyncExitStack 再回退到 SSE，并在最后抛出包含最后一次异常的 ConnectionError。
- MCP 工具名通过 `_sanitize` 去除非字母数字字符并以 server 名前缀拼接，保证多服务器同名工具不会在 ToolRegistry 中冲突。
