测试按被测子系统分层组织：根目录 `test_*.py` 对应 `opc.core.*`、`opc.layer2_organization.*`、`opc.plugins.office_ui.*` 等模块；`tests/cli_board/` 子包专门针对可选依赖 `textual` 的 CLI Board TUI，通过 try/except ImportError + `@unittest.skipIf` 实现可选特性隔离；`tests/e2e/canvas_smoke.mjs` 是独立的 Node.js 脚本，使用 Playwright 启动真实 OPC 服务器进程并驱动 headless Chromium 验证 Canvas/Inspector/Table 视图。共享基础设施集中在 `_temp_paths.py`（提供 `.tmp-test/portable` 下的可移植临时目录上下文管理器）和 `fixtures/native_runtime_v2_suite.json`（供 v2 原生运行时测试复用）。所有用例统一基于 `unittest.TestCase` / `unittest.IsolatedAsyncioTestCase`，异步测试通过 `async def test_*` 编写，外部依赖（LLM、WS、文件系统、配置路径）均通过局部 stub/mock 或 `patch.dict(os.environ)` 注入。
