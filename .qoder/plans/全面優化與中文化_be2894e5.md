## 階段三：建立 i18n 基礎設施

### 3.1 Python 端：建立 `opc/core/i18n.py`
- 輕量級翻譯模組（非 gettext，適合單語言切換場景）
- 核心設計：
  ```python
  # opc/core/i18n.py
  _LOCALE = "zh-TW"  # 預設繁體中文
  _STRINGS: dict[str, dict[str, str]] = {...}
  def t(key: str, **kwargs) -> str: ...
  def set_locale(locale: str) -> None: ...
  ```
- 支援 `OPC_LOCALE` 環境變數切換語言
- 字串集中管理於 `opc/core/locales/zh_TW.py` 和 `opc/core/locales/en.py`

### 3.2 前端：建立 i18n context
- 在 `frontend_src/lib/i18n.ts` 建立簡單翻譯函數
- 字串集中於 `frontend_src/locales/zh-TW.ts` 和 `frontend_src/locales/en.ts`
- 透過 React Context 提供 `t()` 函數
